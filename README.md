Her Jewel - Online Jewelry Store

Project Overview

Her Jewel is an elegant, responsive e-commerce application designed for selling luxury jewelry items. The project demonstrates front-end development skills using HTML, CSS (Bootstrap), and JavaScript. It features a secure login system, an admin dashboard for inventory management, and a functional shopping cart using LocalStorage for data persistence.

Features

User Authentication: Secure Registration and Login system with validation.

Role-Based Access: * Admins (admin@gmail.com) can add new jewelry items.

Customers can view products and add them to their cart.

Dynamic Inventory: Products are stored in the browser and loaded dynamically.

Shopping Cart: Users can add items, view the total cost, and checkout.

Elegant UI: Custom CSS styling with a jewelry-themed background and color palette.

Technologies Used

HTML5

CSS3 & Bootstrap 5

JavaScript (ES6)

LocalStorage (Database simulation)
